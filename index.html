⚙ Stats API Settings (optional)

✅ Nothing to set up here for normal use — the app already uses your verified Torn key with the FFScouter stats API. Just close this and hit Update Stats. The options below are only for power users who want a different stats provider or a CORS proxy.

Advanced overrides

Use {KEY} and {IDS} as placeholders. {IDS} becomes a comma-separated list of target IDs. Default is the FFScouter batch API — you normally never change this.

Leave blank to reuse your verified Torn key. Override only if your stats provider needs a different key.

Rate limit: max 20 requests/minute per IP. One “Update Stats” click is a single batched request (up to 200 IDs); larger lists are auto-chunked and spaced ~3.5s apart, and the button has a short cooldown so it can’t be spammed.
